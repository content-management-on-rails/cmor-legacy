## Installation

Add it to your gemfile:

    # Gemfile
    gem 'cmor-user_area'

Install bundle:

    > bundle install

Install initializer and routes:

    > rails g cmor:user_area:install

Install the migrations and migrate:

    #> rails cmor_user_area:install:migrations
    #> rails db:migrate
