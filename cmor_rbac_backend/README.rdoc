= Cmor::Rbac::Backend

This gem implements a backend for the cmor_rbac gem.

== Motivation

  * Beautiful backend to edit your permissions.

== Installation

Add it to your gemfile:

    # Gemfile
    gem 'cmor_rbac_backend'

Install bundle:

    > bundle install

Install initalizer and routes:

    > rails g cmor:rbac:backend:install

== Usage

Open your browser and visit http://localhost:3000/en/backend/cmor/rbac
