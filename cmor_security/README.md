# Cmor::Security

This module contains seucrity related features of cmor.

## Usage
How to use my plugin.

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'cmor_/_security'
```

And then execute:
```bash
$ bundle
```

## Setup

Generate the initializer:

```bash
$ rails generate cmor:security:install
```
