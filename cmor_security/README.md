# Cmor::Security

This module contains security related features of cmor.

## Usage
How to use my plugin.

## Installation

Add it to your gemfile:

    # Gemfile
    gem 'cmor_security'

Install bundle:

    > bundle install

Install initalizer and routes:

    > rails g cmor:security:install
